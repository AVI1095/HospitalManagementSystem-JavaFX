# Hospital Management System

## Overview
This is a Hospital Management System developed using JavaFX and MySQL.

## Tech Stack
- Java
- JavaFX
- JDBC
- MySQL
- Railway Cloud Database
- NetBeans

## Features
- Role-based login
- Patient management
- Doctor management
- Appointment booking
- Prescription module
- Billing module
- Inventory module
- Audit logs
- Admin stats

## Deployment
- Frontend: JavaFX desktop application
- Backend: Railway cloud MySQL database

## Notes
Internet connection is required to connect to the Railway cloud database.
